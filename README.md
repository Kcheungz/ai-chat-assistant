# AI Chat Assistant (CLI) — Python + OpenAI API

A command-line study assistant that generates:

- explanations of technical topics
- quiz-style questions (with answers + rationales)

## Why this exists

Built as a modular CLI app to demonstrate:

- structured API request handling
- response parsing + validation
- error handling
- maintainable architecture (services/models/prompts/client)

## Setup

### 1) Create virtual environment

Windows:

```bash
python -m venv .venv
.venv\Scripts\activate
```
